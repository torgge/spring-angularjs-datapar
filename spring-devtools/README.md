## Spring DevTools

Adicione a dependência (Maven):

```xml

<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-devtools</artifactId>
    </dependency>
</dependencies>

```

Ou Adicione a dependência (Gradle):

```js

dependencies {
    compile("org.springframework.boot:spring-boot-devtools")
}

```
